# Banco Familiar

Banco ficticio para uso familiar. El dinero dentro de la aplicación no tiene valor real.

## Tecnologías

- Node.js
- Express.js
- SQLite
- Socket.IO
- HTML, CSS y JavaScript puro

## Estado actual

Etapa 1: estructura inicial del proyecto.

La implementación de registro, login, transferencias, préstamos, panel administrador y notificaciones se agregará paso a paso.
